
# PyQt Template

Basic QML based template I use in my projects



## Run Locally

Clone the project

```bash
  git clone https://github.com/Krabbens/pyqt-template.git
```

Go to the project directory

```bash
  cd pyqt-template
```

Install dependencies

```bash
  python -m pip install PyQt5
```

Start the server

```bash
  python main.py
```
